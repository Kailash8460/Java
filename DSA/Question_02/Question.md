## Question 2

Given an array of integers and a target sum, determine if any two numbers sum up to the target.

### Samples

```text
Input 1: [2, 7, 11, 15]
Target = 9
Output: true
Explanation: 2 + 7 = 9.

Input 2: [3, 2, 4]
Target = 6
Output: true
Explanation: 2 + 4 = 6.

Input 3: [1, 2, 3]
Target = 7
Output: false
Explanation: No pair sums to 7.
```
